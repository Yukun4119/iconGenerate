# iconGenerate
Generate icon images for iOS projects

## How to start

* Method 1:

```
pip install iconGenerate
iconGenerate
```

* Method 2:

```
git clone https://github.com/Yukun4119/iconGenerate.git
cd iconGenerate/src/
python iconGenerate.py
```
